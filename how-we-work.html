How we work  
Estimate
Contact us, we'll discuss if our expertise suits your project and provide you with an time and cost estimate.
Analysis
If everything's okay so far, we can sign the contract and dive into detailed analysis of requirements. We can write the requirements.
Development
Once all parties are confident enough with what the result should look like we start the actual development. This can either be planned as major milestones with fixed schedule or in the form of agile iterations, where we do little work, show you intermediate result and you decide where to head next.
Production
Once the MVP is ready we can start deployment and integration. This process is started in parallel usually 1­2 months into the development phase.
Support
After most part of scope has been implemented and the end users can sucessfully user software on everyday basis we switch into support mode making minor fixes and keeping an eye on the production.
